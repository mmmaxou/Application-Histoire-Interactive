Features

Système de sauvegarde :
{
	Plusieurs sauvegardes ( environ 16 )
	Affichage de l'état d'une sauvegarde
	Contenu de la sauvegarde : {
		Choix déjà effectués
		Texte affiché actuel
		Date et Heure
		Temps de jeu
	}
}

Système d'affichage : 
{
	En jeu : {
		Le texte : 
			S’affiche lettre par lettre
		L'image de fond avec éventuellement plusieurs couches
		Les personnages qui parlent
		Moyen d'accéder à l'interface (icône en haut à gauche avec un rouage ? )
Achievment pour les différentes fins (?)
	}
	Interface : {
		Accès aux sauvegardes
		Sauvegarde instantanée
		Accès aux paramètres
	}
}

Paramètres : {
	Taille du texte
	Vitesse de défilement
	Son ( couper/régler le volume / arrêter lors d’un appel)
	Auto-Skip ( Change de dialogue au bout d’un certain temps)
	Vitesse à laquelle le texte apparaît.
	Afficher / Cacher les achievment (?)
}





Contenu secondaire :
Galerie d’images du jeu : {
	Se débloque au fur et à mesure de l’avancement du jeu
Affiche les images du jeu
}
